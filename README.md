# Fish Freshness AI

Sistem klasifikasi spesies dan kesegaran ikan berbasis Artificial Intelligence menggunakan Flask dan TensorFlow.

## Fitur

- Klasifikasi spesies ikan
- Deteksi kesegaran ikan
- Upload gambar ikan
- REST API backend
- Gatekeeper validation

## Teknologi

- Python
- Flask
- TensorFlow
- NumPy
- Pillow

## Struktur Project

- Backend Flask API
- Model AI TensorFlow
- Frontend HTML CSS JavaScript

## Cara Menjalankan

```bash
cd backend
python app.py
